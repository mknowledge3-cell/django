School Papers Portal

A complete membership-based file delivery system for students, built with tiered access, premium downloads, and two dashboards — delivered in just one week.

Project Overview

The client needed a secure system for distributing past papers, with tiered access based on payment level. The estimate for this system from other developers was 2 months — I delivered the full platform in 7 days.

The Challenge

The school needed a digital platform where:

  • Students could access grade-specific papers
  • Premium users could unlock bonus content
  • Admins could easily upload, organize, and manage files
  • Unauthorized file access was prevented
  • Logins needed to be secure and easy

The Solution

I built a Django-powered membership portal with:

  • Tiered access system
  • Real-time admin dashboard
  • Secure file delivery (no direct URLs)
  • Payment-based content unlocking
  • Mobile-friendly UX

Tech Stack

  • Django
  • TailwindCSS
  • PostgreSQL
  • Railway Hosting

Timeline

Estimated by competitors: 8 weeks
Delivered by me: 7 days
